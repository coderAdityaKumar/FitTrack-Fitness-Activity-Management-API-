# Fitness Monolith

Minimal Spring Boot monolith for fitness-related services (backend only).

## Build & run

Using the Maven wrapper:

Unix / macOS
```
./mvnw clean package
./mvnw spring-boot:run
```

Windows (PowerShell / CMD)
```
mvnw.cmd clean package
mvnw.cmd spring-boot:run
```

## Tests

Run unit tests:

```
./mvnw test
```
